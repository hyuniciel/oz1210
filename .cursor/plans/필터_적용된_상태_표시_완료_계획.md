# 필터 적용된 상태 표시 완료 계획

## 현재 상태 분석

### 이미 구현된 기능
1. **활성 필터 뱃지 표시** (이미 구현됨)
   - 활성 필터를 Badge 컴포넌트로 표시
   - 각 뱃지에 개별 제거 버튼 (X 아이콘)
   - 레이블과 값 표시 형식: "지역: 서울", "관광 타입: 관광지" 등

2. **활성 필터 개수 계산 및 표시** (이미 구현됨)
   - `getActiveFilterCount()` 함수로 활성 필터 개수 계산
   - 모바일 토글 버튼에 활성 필터 개수 뱃지 표시
   - 데스크톱 요약 정보 ("N개 필터 적용 중") 표시

3. **개별 필터 제거 기능** (이미 구현됨)
   - `handleRemoveFilter()` 함수로 특정 필터만 제거 가능
   - 활성 필터 뱃지의 X 버튼으로 제거

4. **필터 컨트롤 활성 상태 강조** (부분 구현됨)
   - 지역 필터: 활성 시 `border-primary bg-primary/5` 스타일 적용
   - 정렬 필터: 활성 시 `border-primary bg-primary/5` 스타일 적용
   - 반려동물 필터: 활성 시 `border-primary bg-primary/5` 스타일 적용

### 미구현/개선 필요 항목
1. **관광 타입 필터 활성 상태 강조** (누락됨)
   - 현재 관광 타입 필터 컨테이너에 활성 상태 스타일이 없음
   - 다른 필터와 일관성 있게 활성 상태 강조 필요

2. **선택된 타입 개수 표시** (선택 사항)
   - 관광 타입 필터 레이블 옆에 선택된 타입 개수 표시
   - 예: "관광 타입 (1)" 형식

## 구현 계획

### 1. 관광 타입 필터 활성 상태 강조 추가

#### 1.1 관광 타입 필터 컨테이너 스타일 수정
- **파일**: `components/tour-filters.tsx`
- **위치**: 관광 타입 필터 컨테이너 div (약 490줄)
- **변경 사항**:
  - `currentContentTypeId`가 있을 때 활성 상태 스타일 추가
  - `border-primary bg-primary/5 dark:bg-primary/10` 클래스 적용
  - `cn()` 유틸리티 함수를 사용하여 조건부 스타일 적용
  - 다른 필터 컨트롤과 동일한 스타일 패턴 유지

### 2. 선택된 타입 개수 표시 (선택 사항)

#### 2.1 관광 타입 필터 레이블에 개수 표시
- **파일**: `components/tour-filters.tsx`
- **위치**: 관광 타입 필터 레이블 (약 477줄)
- **구현 내용**:
  - 선택된 타입이 있으면 레이블 옆에 개수 표시
  - 예: "관광 타입 (1)" 형식
  - 선택된 타입이 없으면 개수 숨김

### 3. 기능 검증 및 완료 체크

#### 3.1 모든 필터 컨트롤의 활성 상태 강조 확인
- 지역 필터: 활성 상태 강조 확인
- 관광 타입 필터: 활성 상태 강조 추가 확인
- 정렬 필터: 활성 상태 강조 확인
- 반려동물 필터: 활성 상태 강조 확인

#### 3.2 활성 필터 표시 기능 확인
- 활성 필터 뱃지 표시 확인
- 활성 필터 개수 표시 확인
- 개별 필터 제거 기능 확인

## 구현 세부사항

### 관광 타입 필터 활성 상태 강조 코드

```tsx
// 현재 코드
<div className="flex flex-col gap-2 max-h-32 overflow-y-auto border rounded-md p-2">

// 변경 후
<div className={cn(
  'flex flex-col gap-2 max-h-32 overflow-y-auto border rounded-md p-2',
  currentContentTypeId && 'border-primary bg-primary/5 dark:bg-primary/10',
)}>
```

### 선택된 타입 개수 표시 코드

```tsx
// 현재 코드
<Tag className="h-4 w-4" aria-hidden="true" />
관광 타입

// 변경 후
<Tag className="h-4 w-4" aria-hidden="true" />
관광 타입
{selectedTypeIds.length > 0 && (
  <span className="ml-1 text-xs text-muted-foreground">
    ({selectedTypeIds.length})
  </span>
)}
```
